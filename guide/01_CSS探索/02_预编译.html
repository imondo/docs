CSS Module #

在页面中声明的类名都会根据当前页面，打一个唯一的 hash 值，比如我们页面中声明的 className={style.form}，最终在浏览器中显示的是这样的：

与另外两者的深度选择器想对比，CSS Module 有 :global 这个关键词

less
  .form {
    // css Module 
    :global {
      .za-cell {
        color: red;
      }
    }
  }
  • less
less
.form {
	/deep/ .za-cell {
		color: red;
	}
	::v-deep .za-cell {
		color: red;
	}
}
  • scss
scss
.form {
	/deep/ .za-cell {
		color: red;
	}
	::v-deep .za-cell {
		color: red;
	}
}
