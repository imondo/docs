# Array循环性能问题

对于数据量`> 1000`的数组操作

* 大量数据可先定义缓存数组`arr`长度，再循环添加元素

* 优先使用`for i++`循环

* 赋值时注意缓存

```
let postData = [];
let _projectId = $('[name="projectId"]').val();
let _rogId = $('[name="orgId"]').val();
postData.length = tableData.length;
console.log(new Date());
for (let i = 0, len = tableData.length; i < len; i++) {
  postData[i] = {
    projectId: _projectId,
    id: tableData[i].id
    req: { orgId: _rogId }
  };
}
```
