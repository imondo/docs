# 大屏解决方案

解决方案使用 `transform` 来缩放。

计算好相应的倍数
```javascript
// 获取 window 的高度
var clientH = window.innerHeight;

// 初始 window 与 body 的比值
var ratio = clientH / document.body.clientHeight;
```
    
使用动画进行按比例缩放

```javascript
transform: 'scale(' + ratio + ')'
```
