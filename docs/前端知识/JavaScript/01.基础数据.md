# 基础数据

## 分别有哪些类型

* 基础类型

  * String：字符串类型
  * Number：数字类型
  * Boolean：true，false
  * undefined：undefined
  * null：null

* 对象（引用）类型

  * Object：任意对象
  * Function：可以执行的函数对象
  * Array：有序的数值下标的数组对象

## 怎么判断类型

* typeof

```
typeof 'Mondo' // "string"
typeof 1 // "number"
typeof true // "boolean"
typeof false // "boolean"
typeof null // "object"
typeof undefined // "undefined"
```
